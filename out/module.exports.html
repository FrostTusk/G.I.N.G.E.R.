exports(event)

This function will output a given event into an easily parsed log entry. This output will make use of console.log.

Constructor

new exports(event)

Parameters:
Name Type Description
event The only parameter for this method, we only use 1 object because this allows for swift changes in the way we structure events. This object will include all the neccessary data for logging. All this data is optional.
Properties
Name Type Description
message This attribute is simply the message that should be added to the event. The default is undefined.
source This attribute can take multiple values. If no source was provided, the source will be 'source: unkown source'.
tags An optional attribute to allow for more detailed filtering of the logs. The default is [].
